# 🚀 Basic Token Creator on Stellar (Soroban)

## 📌 Project Description
The **Basic Token Creator** is a simple smart contract built on the Stellar network using Soroban. It enables users to create and manage a custom token with essential functionalities like minting, transferring, and tracking balances.

This project demonstrates how decentralized token systems can be built efficiently using Soroban smart contracts.

---

## ⚙️ What It Does
- Initializes a token with an admin account
- Allows the admin to mint new tokens
- Enables users to transfer tokens to others
- Stores and retrieves user balances
- Tracks total token supply on-chain

---

## ✨ Features
- 🪙 Token creation and management
- 🔐 Admin-controlled minting
- 🔄 Peer-to-peer token transfers
- 📊 Balance tracking for all users
- ⚡ Fast and low-cost transactions (Stellar network)
- 🧩 Simple and extensible design

---

## 🧠 Use Cases
- Custom cryptocurrency
- Reward or loyalty points system
- In-app digital currency
- Educational blockchain projects

---

## 🛠 Tech Stack
- Rust
- Soroban SDK
- Stellar Blockchain

---
